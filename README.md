## ✨ Overview

FTP File Manager is a modern, high-performance web interface designed for managing files on your server. It features a beautifully crafted UI, seamless interactions, and robust file management capabilities.

<br />

## 🔥 Key Features

- 🔐 **Secure Access:** Password-protected login area with a sleek, modern UI.
- 📱 **Fully Responsive:** Optimized for both Desktop and Mobile devices.
- ☁️ **File Uploads:** Fast and smooth file uploading experience.
- 📋 **Detailed Insights:** View file sizes, types, and last modified dates in a clean table format.
- 🔗 **Shareable Links:** 1-click URL copying directly to your clipboard.
- 🗑️ **Easy Deletion:** Safely delete files with a confirmation modal.
- 📊 **Dashboard Stats:** Quick overview of total files, folders, and storage space used.

<br />

## 🛠️ Technology Stack

- **Framework:** React 18
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Build Tool:** Vite

## 🚀 Quick Start / Installation

Follow these steps to get the project running locally:

### 1. Clone the repository
```bash
git clone https://github.com/rasheddevx/FTP-File-Manager.git
cd FTP-File-Manager
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the development server
```bash
npm run dev
```
